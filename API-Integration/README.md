# Weather Dashboard

A simple weather dashboard developed using **HTML, CSS, and JavaScript**. This project uses API integration to fetch and display real-time weather information based on the city entered by the user.

## Features

* Search weather using city name
* Real-time weather information
* City suggestions while typing
* Displays temperature, humidity, and wind speed
* Responsive design
* Error handling for invalid input
* Dynamic weather updates

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Open-Meteo API

## Project Files

├── index.html
├── style.css
├── script.js
└── README.md

## How to Run

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in any web browser

## API Used

This project uses the **Open-Meteo API** to fetch real-time weather data.
